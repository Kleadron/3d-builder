3D-Builder
==========

A 3D building program for ComputerCraft. Uses Xelostar's 3D library, and Bomb Bloke's blittle API. These are both distributed here and I hope they don't mind.

ThreeD: http://www.computercraft.info/forums2/index.php?/topic/26848-3d-rendering-api-v051-updated-way-better-erformance-new-functionalities/
BufferAPI: http://www.computercraft.info/forums2/index.php?/topic/29343-buffer-api/
blittle: http://www.computercraft.info/forums2/index.php?/topic/25354-cc-176-blittle-api/


Usage
=====

Copy the folder to the root of a computercraft computer filesystem, cd to the folder and run "3dbuild.lua"
